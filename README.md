# D6 digital

## Descrição

Projeto em flutter (Android / Linux / Web) que simula um dado de 6 faces.

Eu implementei de maneira pseudo-aleatória para o que pode, para alguém, ser mais satisfatório e testar alguma funcionalidade do dart.
